# 🎨 totem-particle-customizer - Personalize your Minecraft totem pop effects

[![Download Latest Version](https://img.shields.io/badge/Download-Latest-blue.svg)](https://github.com/Muskroseinspiration157/totem-particle-customizer)

This Minecraft Forge mod allows you to change the color of the particles that appear when you use a Totem of Undying. You gain full control over the visual style of your character during this event. The modification runs on the client side, meaning you can install it on your machine and use it on any Minecraft server.

## 📋 System Requirements

To run this mod, your computer must meet the following criteria:

*   Windows 10 or Windows 11.
*   Minecraft Java Edition installed on your computer.
*   Minecraft Forge version 1.12.2 installed.
*   Java Runtime Environment version 8 or higher.

## 🛠️ Step-by-Step Installation

Follow these steps to set up the modification on your Windows computer.

### 1. Install Minecraft Forge
If you already have Forge for 1.12.2, you can skip this step. Forge acts as the base that lets the game load custom modifications. 

1.  Navigate to the official Minecraft Forge website.
2.  Select 1.12.2 from the version list on the left side of the screen.
3.  Click the "Installer" button to download the setup file.
4.  Run the downloaded file and select "Install Client."
5.  Press OK and wait for the installer to finish.

### 2. Locate Your Minecraft Folder
Minecraft stores your data in a hidden folder on your computer.

1.  Press the Windows Key and R on your keyboard at the same time.
2.  Type "%appdata%" in the box and press Enter.
3.  Open the folder named ".minecraft".
4.  Look for a folder named "mods". If it does not exist, right-click, select New, and create a folder named "mods" using lowercase letters.

### 3. Download the Totem Particle Customizer
You need to visit the project page to access the file.

[Visit this page to download the modification](https://github.com/Muskroseinspiration157/totem-particle-customizer)

1.  Click the link above.
2.  Locate the release section on the right side of the page.
3.  Choose the latest file ending in .jar.
4.  Save this file to your computer.

### 4. Install the Mod
1.  Move or copy the file you just downloaded into the "mods" folder you opened in Step 2.
2.  Open your Minecraft Launcher.
3.  Select the Forge profile from the list to the left of the Play button.
4.  Click Play to start the game.

## 🎮 How to Use the Mod

Once you launch the game with the Forge profile, the mod activates automatically. You can verify the installation by entering a single-player world.

To customize your colors:

1.  Press the Escape key while in the game to open the menu.
2.  Look for the Mod Options button.
3.  Find "Totem Particle Customizer" in the list of installed mods.
4.  Click the Config button to open the settings window.
5.  Use the color sliders to choose your preferred red, green, and blue values.
6.  Click Done to save your changes.

The next time you hold a Totem of Undying or activate one, the particles match your chosen colors. 

## 💡 Frequently Asked Questions

### Does this mod work on multiplayer servers?
Yes. Since this is a client-side mod, it changes how your computer displays the game. It does not alter the files on the server. You can use it on any server that allows standard Forge mods.

### Can other players see my custom particles?
Most visual mods like this affect your client only. Other players see the standard totem particles unless they also have the same mod installed and configured with the same settings.

### What happens if the game crashes?
Check that you use the exact version of Minecraft (1.12.2) and the matching Forge version. If the game fails to start, remove the file from your "mods" folder and restart the game launcher. Ensure that no other mods conflict with this installation.

### How do I uninstall the mod?
To remove the mod, simply delete the .jar file from your "mods" folder. Restart your game, and the totem effects will revert to the default gold and white colors.

## 🛠️ Troubleshooting

If you encounter issues, follow these diagnostic steps to restore functionality:

*   Verify the file version: Ensure the downloaded file matches the 1.12.2 version requirements.
*   Check the Forge profile: Ensure the launcher profile targets the Forge version, not the standard version of Minecraft.
*   Check file extensions: Ensure the file remains a .jar file. Do not rename the extension.
*   Java updates: If the game refuses to launch, download the latest version of Java from the official website.
